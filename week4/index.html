    Inherit all the parent class qualities.

  • Display

    When you want to show something to a user, you use a section (see Section 1) and the special word ‘innerHTML.’

  • Property Descripter

    value: the value of the property and is undefined by default

  • Property Descripter

    configurable: boolean value shows whether you can delete a property or change any of its attributes, and is false by default.

  • Property Descripter

    writable: boolean value shows whether a property can be changed or not, and is false by default.

  • Property Descripter

    enumerable: boolean value shows whether a property will show up when the object is displayed in a for in loop, and is false by default
