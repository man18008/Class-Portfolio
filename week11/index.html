Notes for Week 2

typeof will find out the type of a value (ie if it's boolean, string, number, object)

parseInt converts a string back into a number. It needs a parameter of how many numbers you want it to show. (Chapter 2)

In an array, array.length - 1 will give you the value of the last item in the array. You have to subtract one because the array starts at 0.

Sets do not have index notation for inspecting individual entries, so you can't find the value of the first element in a set.

You can also assign a function to a variable like this: const nameOfVariable = function nameOfFunction() {console.log ('Message');}
